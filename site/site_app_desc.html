App description


This app is a web scraper that scrapes event information from various ticket websites in Japan.
The app is designed to scrape event information from the following websites:

  • t.pia.jp
    •
  • eplus.jp
    •
  • l-tike.com
    •

The app allows Users to select which websites they want to scrape and provides settings options for eplus.jp and l-tike.com.
(Data for t.pia.ja is taken from its front page, therefore cannot be modified.)
Currently, the User can choose the scraped months for eplus.jp and precise dates for l-tike.com.
(Currently it checks only for Kanto/Koshinetsu/Kinki/Hokuriku regions - will probably change in the future.)
