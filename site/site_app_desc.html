🎟️ App description 🎟️

This app is a web scraper that scrapes event information from various ticket websites in Japan. The app is designed to scrape event information from the following websites:


  • ✨Ticket Pia (t.pia.jp)
  • ✨e+ (eplus.jp)
  • ✨Lawson Ticket (l-tike.com)
  • ✨ZAIKO (zaiko.io)

The app allows Users to select which websites they want to scrape and the date range.
